### Pi Trainer


A lightweight pi trainer written in PHP

![alt tag](http://www.righteousbanana.com/images/pi-trainer.png)



## Getting Started

1. Copy the bundle into a location served by a web server
2. Launch <your-domain>/pi-trainer
